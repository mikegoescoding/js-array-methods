# JavaScript Array Methods

Practice exercises for JavaScript's core array methods — written as part of a structured full-stack upskilling plan.

## What I practiced

- `map()` — transforming every element into a new array
- `filter()` — keeping only elements that pass a condition
- `reduce()` — collapsing an array into a single value (number, string, or object)
- `find()` — returning the first match
- `some()` / `every()` — boolean checks across an array
- `forEach()` — looping without returning a new array
- Arrow function syntax — converting between long form, arrow, and shorthand
- Ternary operator — shorthand if/else for single expressions
- Chaining methods — `filter().map()` and `filter().reduce()`

## Files

| File | Description |
|------|-------------|
| `map.js` | Transformation exercises including the missing return bug |
| `filter.js` | Filtering exercises including chaining with map |
| `reduce.js` | Three exercises: sum, longest word, count occurrences |
| `other-methods.js` | find, some, every, forEach on a students dataset |
| `syntax-reference.js` | Arrow function conversion rules and ternary reference |

## How to run

```bash
node map.js
node filter.js
node reduce.js
node other-methods.js
```
